Second Mag Website (site de anunturi)

Acest site a fost creat pentru XAMPP

1)Folderul cu "SecondMag" se va pune in folderul "htdocs" unde a fost instalat XAMPP

2)Din MySQLi se va importa baza de date "nameless.sql"

3)In "auth.php" se vor introduce datele unui Gmail pentru PHP Mailer

4)In "const.php" se pot modifica constante si se poate activa si dezactiva verificare de email si captcha (Captcha merge doar pe localhost)

5)Pagina principala se acceseaza la adresa "localhost/SecondMag/mage.php"
